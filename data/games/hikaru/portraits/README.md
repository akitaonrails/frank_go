# Character portraits (optional, user-supplied)

Drop images here to replace the default go-stone medallions shown during Hikaru
no Go study sessions. Name each file after the character in kebab-case; `png`,
`jpg`, `jpeg` and `webp` work:

    sai.png
    hikaru-shindo.jpg
    akira-toya.png
    toya-koyo.png
    tetsuo-kaga.png
    seiji-ogata.png
    kuwabara-honinbo.png
    yoshitaka-waya.png
    kosuke-ochi.png
    tatsuhiko-kadowaki.png
    ko-yongha.png
    the-rival-tutor.png

Square-ish images look best (they are shown as small circles). Nothing in this
folder is committed to the repository or redistributed — the manga's character
artwork is copyrighted, which is why the app does not bundle it.
